Bone Throne

A top down roguelike action game set in a cartoonish wild west desert. Game was created with a homemade component based game engine developed in C++ using SDL2 for game rendering. Players navigate with wasd, aim with cursor, and dodge roll with space. Defeat all the enemies on each screen to advance. Uses open Nuclear Throne art assets.
